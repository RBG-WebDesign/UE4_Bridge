# Skill: edit_project_config_and_input

Write action and axis mappings to DefaultInput.ini and edit project config files.

## DefaultInput.ini location
```
<ProjectDir>/Config/DefaultInput.ini
```

## Adding action mappings
Append to the `[/Script/Engine.InputSettings]` section:
```ini
+ActionMappings=(ActionName="Jump",bShift=False,bCtrl=False,bAlt=False,bCmd=False,Key=SpaceBar)
+ActionMappings=(ActionName="Fire",bShift=False,bCtrl=False,bAlt=False,bCmd=False,Key=LeftMouseButton)
```

## Adding axis mappings
```ini
+AxisMappings=(AxisName="MoveForward",Scale=1.000000,Key=W)
+AxisMappings=(AxisName="MoveForward",Scale=-1.000000,Key=S)
```

## Python append pattern
```python
import os
project_dir = unreal.Paths.project_dir()
ini_path = os.path.join(project_dir, "Config", "DefaultInput.ini")
with open(ini_path, "a", encoding="utf-8") as f:
    f.write("\n; Generated by PromptBrush\n")
    f.write('+ActionMappings=(ActionName="Jump",...,Key=SpaceBar)\n')
```

## Pipeline function
`handlers/promptbrush.py` -- `_write_input_mappings(action_mappings, axis_mappings, project_dir)`

## Key names reference (common)
SpaceBar, LeftMouseButton, RightMouseButton, W, A, S, D, Left, Right, Up, Down,
X, Z, C, V, F, E, Q, R, Escape, Tab, Enter, LeftShift, LeftControl

## Notes
- Appending is safe; UE4 reads all matching keys in the ini
- Do not duplicate mappings -- check if already present before appending
- Reload input settings in editor: Edit > Project Settings > Input (not needed at startup)
